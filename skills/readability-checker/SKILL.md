# 카드뉴스 가독성 체크리스트

## 평가 방법
렌더링된 PNG를 슬라이드별로 검토하여 각 항목 점수(1~10) 산출.
**5점 이하 항목은 반드시 수정 요청.**

---

## 슬라이드별 평가 항목

### 1. 텍스트 밀도 (배점: 20점)
모바일(375px 기준) 환경 시뮬레이션:

| 기준 | 통과 | 주의 | 수정 필요 |
|------|------|------|-----------|
| 본문 줄 수 | 4줄 이하 | 5줄 | 6줄 이상 |
| 커버 제목 글자 수 | 10자 이하 | 11~13자 | 14자 이상 |
| 헤드라인 글자 수 | 15자 이하 | 16~18자 | 19자 이상 |
| 한 줄 글자 수 | 20자 이내 | 21~24자 | 25자 이상 |

체크 방법: 슬라이드를 스마트폰 화면(가로 375px)으로 상상하고 텍스트가 잘리거나 답답한지 확인.

### 2. 폰트 위계 명확성 (배점: 20점)
3단계 위계가 시각적으로 명확히 구분되는가:

- **제목**: 가장 크고 굵음 → 가장 먼저 눈에 들어와야 함
- **소제목/배지**: 중간 크기, 색상으로 구분
- **본문**: 가장 작고 가벼운 무게

확인 체크:
- [ ] 3초 안에 제목이 무엇인지 파악 가능한가?
- [ ] 본문이 제목보다 눈에 띄지 않는가?
- [ ] 강조(highlight)가 1~2개 이내인가?

### 3. 색상 대비 (배점: 20점)
WCAG AA 기준:

| 텍스트 종류 | 최소 대비율 | 체크 방법 |
|------------|------------|-----------|
| 일반 본문 (18px 미만) | **4.5:1 이상** | 배경색 vs 텍스트 색 |
| 대형 텍스트 (18px 이상 또는 Bold 14px) | **3:1 이상** | 배경색 vs 텍스트 색 |
| 배지/태그 텍스트 | 4.5:1 이상 | 배지 배경 vs 텍스트 |

주의할 조합:
- ❌ 밝은 노란색 배경 + 흰색 텍스트
- ❌ 다크 배경 + 회색 텍스트
- ✅ 다크 배경 + 흰색/밝은 텍스트
- ✅ 흰색 배경 + 진한 텍스트

### 4. 3초 가독 테스트 (배점: 20점)
슬라이드를 3초간 본다고 상상:

- [ ] 핵심 메시지가 1개인가? (2개 이상이면 감점)
- [ ] 가장 먼저 눈에 들어오는 요소가 핵심 메시지인가?
- [ ] 다음 슬라이드로 넘어가고 싶은 궁금증이 생기는가?
- [ ] 불필요한 장식 요소가 메시지를 방해하지 않는가?

### 5. 카드 간 시각적 흐름 (배점: 10점)
전체 슬라이드를 순서대로 검토:

- [ ] 1장(커버) → 2장 전환이 자연스러운가?
- [ ] 각 카드의 마지막 문장이 다음 카드를 암시하는가?
- [ ] 마지막 본문 → CTA 연결이 논리적인가?
- [ ] 전체 색상·스타일이 일관적인가?

### 6. CTA 명확성 (배점: 10점)
마지막 CTA 슬라이드 전용:

- [ ] 행동을 유도하는 동사로 시작하는가? (예: "지금 확인", "댓글로 공유")
- [ ] CTA 버튼/텍스트가 다른 요소보다 시각적으로 강조되어 있는가?
- [ ] 해시태그 3개가 포함되어 있는가? (#잡코리아오카방 필수)
- [ ] 계정명이 표기되어 있는가?

## 평가 결과 출력 형식

```
[가독성 검토 결과]

슬라이드별 점수:
1장(cover)        : 8/10 — 제목 길이 적절, 여백 충분
2장(content-stat) : 6/10 — ⚠️ 본문 6줄, 4줄 이하로 줄일 것
3장(content-list) : 9/10 — 양호
4장(content-steps): 7/10 — 단계 제목 각 12자, 10자 이하 권장
5장(content)      : 5/10 — ⚠️ 대비율 부족, 텍스트 색상 변경 필요
6장(cta)          : 9/10 — CTA 명확, 해시태그 포함

종합 평가: 7.3/10
수정 필수: 2장(텍스트 밀도), 5장(색상 대비)
수정 권장: 4장(헤드라인 길이)
```

## 자주 발생하는 가독성 문제

| 문제 | 원인 | 해결 방법 |
|------|------|-----------|
| 텍스트가 너무 많음 | 내용을 다 담으려는 욕심 | 핵심 1개만 남기고 나머지 삭제 |
| 제목이 잘 안 보임 | 배경과 대비 부족 | 텍스트 색상 또는 배경 변경 |
| 슬라이드가 산만함 | 강조 요소 과다 | highlight를 1개로 줄임 |
| 흐름이 끊김 | 각 카드가 독립적으로 구성됨 | 이전 카드 내용을 받아서 시작 |
| CTA가 약함 | 수동적 문장 | 동사로 시작하는 능동적 문구로 교체 |
